# 5文型

2016/07/07

パン屋の復習。

英語の基本形。S=Subject, V=Verb, O=Object, C=Complement

1. S + V
1. S + V + C
1. S + V + O
1. S + V + O + O
1. S + V + O + C

## S+V

```
I live in Akihabara
S  V   (           )
```

## S+V+C

SとCがイコールでつながる、と考えればよい。

```
He is 19
S  V  C
```

## S+V+O

```
He loves you
S    V   O
```

## S+V+O+O

```
Give me chocolate
 V   O   O
```

## S+V+O+C

OとCがイコールでつながる、と考える。

```
When I was young, friend called me Take-chan.
   (Sub cause)      S      V    O     O
```
